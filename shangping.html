1.查询所有商品的平均价格:mysql> select avg(price) as tatal from commodity;
2.查询某个一类型商品的平均价格:mysql> select avg(price) as tatal from commodity where id=1;
3.查询某个商品的存货价值:mysql> select cname,price*number as value from commodity where id=3;
4.查询某个订单的金额:mysql> select commodity.cname,commodity.price*orders.sunmber from commodity,orders where commodity.cid=orders.oid;
5.查询价格最低三个商品:mysql> select cname,price from commodity order by price limit 0,3;
6.查询最新添加的一个商品:mysql> select cid,cname,price from commodity order by cid desc limit 0,1;
7.查询商品的所有信息以及它的类型:mysql> select * from commodity,type where commodity.id=type.id;
8.查询销售最高的三个商品:mysql> select commodity.cname,orders.sunmber-commodity.number as cha from commodity as commodity,orders as orders where commodity.cid =orders.id group by commodity.cname limit 0,3;
